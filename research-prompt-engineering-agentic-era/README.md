## Key Findings

- **Few-shot examples** improved accuracy from 0% to 90% in medical coding (no model upgrade needed)
- **Three sentences** added to an agentic system prompt yielded ~20% improvement on SWE-bench
- **Role prompting** has little to no effect on correctness --- context and examples are far more impactful
- **Context engineering** (curating optimal tokens across all sources) is replacing prompt engineering as the dominant framework
- **Modern models** (Claude 4.6, GPT-5.x) are more instruction-following, meaning previously aggressive prompting now causes overtriggering
