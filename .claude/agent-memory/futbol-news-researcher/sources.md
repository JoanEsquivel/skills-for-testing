# Source Reliability Notes

## Tier 1 - Consistently Excellent
- **ESPN**: Best for match reports, scores, structured data. Always has game analysis pages.
- **Barca Blaugranes**: Daily news roundups with multiple stories. Great for Barcelona-specific angles.
- **UEFA.com**: Official fixtures, draw results, inside stories. Authoritative for Champions League.
- **Official club sites** (fcbarcelona.com, realmadrid.com): Press conferences, official injury confirmations.

## Tier 2 - Good for Specific Angles
- **Al Jazeera Sports**: Good live blogs and match previews. Free access.
- **beIN Sports**: Champions League previews and bracket info.
- **Sky Sports**: Transfer Centre is the best single page for transfer rumors.
- **Heavy.com / CBS Sports**: Good injury reporting with context.
- **Planet Football / OneFootball**: Stat comparisons and record tracking.
- **Goal.com**: Good for reaction pieces and listicles.

## Tier 3 - Use With Caution
- **TEAMtalk**: Transfer rumors can be speculative; verify with second source.
- **Football-Talk**: Aggregates rumors, not always sourced.
- **Yahoo Sports**: Syndicated content, sometimes delayed.

## Notes
- Transfermarkt landing page rarely gives usable search results; search for specific player/club instead.
- TikTok search results are mostly links to discover pages, not specific trend data. Better to search for "viral football" + platform name.
- Reddit/fan forum content doesn't surface well in web search; would need direct page fetching.
