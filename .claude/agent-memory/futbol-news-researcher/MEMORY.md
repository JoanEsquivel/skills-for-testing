# Futbol News Researcher - Agent Memory

## Source Reliability
- See [sources.md](sources.md) for detailed notes on source quality

## Research Workflow
- Run 5+ parallel WebSearches in the first pass (one per source category)
- Follow up with 4-5 targeted searches on the strongest stories
- ESPN, Barca Blaugranes, and official club sites give the best structured match data
- Transfer rumors: Sky Sports Transfer Centre and TEAMtalk have the freshest rumors
- For stat comparisons (e.g., Yamal vs Messi at same age), PlanetFootball and OneFootball have good structured data

## Content Patterns
- "Win the battle, lose the war" narratives (e.g., Barca winning 3-0 but eliminated) perform well as shock hooks
- Injury crisis stories work best when you stack the count (e.g., "SEVEN ACLs in three years")
- Goalkeeper scoring goals is always viral -- timeless content
- Ryan Reynolds / Wrexham stories bridge casual and hardcore football audiences
- Stat comparison hooks (Yamal vs Messi vs Ronaldo at same age) are evergreen and highly shareable

## Format Notes
- 30s scripts work for single-moment highlights (one goal, one stat)
- 45s scripts work for stat-driven or comparison content
- 60s scripts needed for multi-thread stories (injury crisis, match recaps with context)
- Spanish recommended for La Liga-specific content targeting Spanish-speaking audiences
- Bilingual works well for Barca content (global + Spanish fanbase)
