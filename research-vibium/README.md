## What is Vibium?

Vibium is an open-source, AI-native browser automation framework created by **Jason Huggins** — the original co-creator of Selenium (2004) and Appium (2012), and co-founder of Sauce Labs. Its tagline: *"Browser automation without the drama."*

It is designed as the successor to Selenium for the AI era, enabling both human developers and AI agents to control browsers through natural language commands, CLI tools, MCP servers, and language-specific SDKs.

## Key Takeaways

1. Created by the same person who built Selenium — this is not a random startup
2. Built on W3C WebDriver BiDi (open standard), not Chrome DevTools Protocol
3. AI-first: MCP server integration, natural language test authoring, self-healing locators
4. Available on npm and PyPI, Apache 2.0 license
5. Still early-stage — great for experimentation, not yet production-ready for enterprise
6. V2 (Retina + Cortex) will add recording, app mapping, and intelligent planning
